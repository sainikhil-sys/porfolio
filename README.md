# Sai Nikhil — Portfolio<br><br>
url:https://sn-portfolioo.netlify.app/

Personal portfolio site built with Vite, React, TypeScript, Tailwind CSS, and shadcn/ui.

## Local development

Requires [Node.js](https://nodejs.org/) (LTS recommended) and npm.

```sh
git clone <your-repo-url>
cd sai-nikhil-portfolio
npm install
npm run dev
```

The dev server runs at `http://localhost:8080` by default.

## Scripts

| Command        | Description              |
| -------------- | ------------------------ |
| `npm run dev`  | Start dev server         |
| `npm run build`| Production build         |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint               |
| `npm test`     | Run tests                |

## Resume

Place your PDF at `public/resume.pdf`. The hero **Download Resume** button serves that file.
